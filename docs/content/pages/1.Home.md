### Pain & Resonance Section

#### 콘텐츠 내용

**Pain 포인트 5개**

| #   | Pain 포인트                            | 감정 키워드            |
| --- | ----------------------------------- | ----------------- |
| 1   | 지금 돈 쓰는 게 맞나, 괜히 날릴까 불안해요           | 불안 / 돈낭비 / 망설임    |
| 2   | 처음부터 제대로 하고 싶은데, 뭘 어떻게 시작해야 할지 막막해요 | 막막 / 막연함 / 방향상실   |
| 3   | 일단 홈페이지만 있으면 매출이 생기겠죠?              | 착각 / 기대 / 허탈      |
| 4   | 가격은 올리고 싶은데, 그만한 이유를 설명하기 어려워요      | 설명불가 / 설득안됨 / 답답함 |
| 5   | 조금만 더 알아보자 하다가, 시간만 계속 지나가요         | 미룸 / 무기력 / 제자리    |
**Pain 포인트와 공명 문장잇는 브릿지 문장**

```
나는,
당신은,
우리는 지금 설명 가능한가요?
```

**공명 문장**

```
의미를 부여하면 의미가 됩니다.
의미가 있으면 선명해집니다.
선명해지면 설명이 가능해지고,
설명이 가능해지면 연결됩니다.
```

### For Who Section

**섹션 상단 문장**
```
이런 전환의 순간에 함께합니다.
```

**타겟 5개**

| #   | 타겟           | 전환         | 설명                                       | 관계 철학                                |
| --- | ------------ | ---------- | ---------------------------------------- | ------------------------------------ |
| 1   | 예비창업가        | 아이디어 → 브랜드 | 아이디어는 있지만, 어디서부터 어떻게 해야할지 막막한 분          | 첫 브랜딩의 막막함을 함께 정리해가는 멘토              |

| 5   | 스타트업         | MVP → 브랜드  | 투자자와 고객 모두에게 설명가능하고 확장 가능한 브랜드 구조가 필요한 팀 | 빠르게 검증하고 단단하게 쌓아가는 빌더                |

### Brand Story Section

#### 콘텐츠 내용


```
1.
   "저는 늘 경계에 있었습니다.
   
   호주 Macquarie 대학에서 IT를 공부했습니다.
   한국인도 호주인도 아닌 경계에서 
   제 정체성을 계속 설명해야 했습니다.
   
   그래서 압니다. 
   설명되지 않으면 연결되지 않는다는 것을."


2.
   "IT, 방송, 요리, 목수, PM, 디자인 등
   서로 다른 일을 오래 건너오며
   하나를 배웠습니다.
   사람이 이해하고 선택하는 방식은 닮아 있다는 것을.
   
   그리고 이 모든 경험이
   당신의 사업을 이해하는 힘이 되었다는 것을 이제 압니다.
   
   그래서 당신의 언어와 고객의 언어 사이, 
   그 경계에서 브랜드를 번역합니다."

3.
   "감성 위에 구조를 더하고, 
   구조 안에 감성을 불어넣는 일.
   
   보이지 않는 분위기를 언어로 정리하고,
   흐릿한 가치를 작동하는 구조로 바꾸는 일.
   
   당신의 고유함을 기록하고
   고객과 연결하는 이야기로 만드는 일."

4.
   "불안을 끄고, 기준을 켭니다.
   
   이제 브랜드를 켤 차례입니다."
   
```

### Brand Diagnosis Section

#### 콘텐츠 내용

**진입**
```
> 브랜드 켜는 중...
> 브랜드 켜는 중......  [커서 깜빡임]
> 브랜드 진단을 시작하시겠습니까?

[ YES ]  [ NO ]
```

**YES 선택 시**
```
> initializing brand_diagnosis...
> loading user_profile...
> connecting to NEMO:ON...
> ████████████████ 100%
> COMPLETE ✓
→ 브랜드 진단 페이지 자동 이동
```

**NO 선택 시**
```
> ████████████████ (중간에 멈춤)
> ERROR 404: brand_clarity not found
> 브랜드 진단이 필요합니다.
> redirecting...
→ 브랜드 진단 페이지 자동 이동
```
