
#### 콘텐츠 내용(진단구조)

2단계 스텝.

### 상단 타이틀

```
당신의 브랜드는 지금 설명 가능한가요?
간단한 진단으로 현재 브랜드 상태를 확인해보세요.
프로그레스 바 (진행률 %)
```


### 브랜드 정보 체크 질문/보기

**1단계: 지금 브랜드 상태를 알려주세요**

Q1. 어떤 분야의 비즈니스인가요? (단일선택)
```
테크/IT / 헬스케어 / 교육 / 리테일/이커머스
음식/F&B / 패션/뷰티 / 부동산 / 금융 / 기타
```

Q2. 당신의 브랜드는 어떤 느낌으로 기억되길 바라나요? (다중선택)
```
신뢰할 수 있는 / 혁신적인 / 친근한 / 전문적인
럭셔리한 / 심플한 / 역동적인 / 안정적인
```

Q3. 당신의 브랜드와 가장 잘 어울리는 색상 계열은 무엇인가요? (단일선택)
```
블루 계열 (신뢰, 안정) / 그린 계열 (자연, 성장)
레드 계열 (열정, 에너지) / 오렌지 계열 (활동, 따뜻함)
퍼플 계열 (프리미엄, 창의) / 블랙/그레이 계열 (모던, 심플)
기타/잘 모르겠음
```

**2단계: 지금 가장 고민되는 지점을 알려주세요**

Q4. 지금 당신의 브랜드는 어느 단계에 있나요? (단일선택) ← 결과 분류 기준
```
브랜드가 전혀 없음 → 신생 브랜드형
로고만 있음 → 신생 브랜드형
기본적인 아이덴티티 있음 → 성장 브랜드형
어느 정도 완성되었지만 개선 필요 → 성장 브랜드형
완성도 높지만 차별화 필요 → 완성 브랜드형
```

Q5. 가장 큰 고민은 무엇인가요? (다중선택)
```
브랜드 정체성이 불분명함
경쟁사와 차별화 어려움
타겟 고객에게 어필하지 못함
일관성 있는 브랜드 경험 부족
온라인에서의 브랜드 인지도 부족
브랜딩 예산 대비 효과 미흡
```

Q6. 지금 고객들은 당신의 브랜드를 어떻게 받아들이고 있나요? (단일선택)
```
아직 시작 전이에요
브랜드를 잘 기억하지 못함
비슷한 경쟁사와 헷갈려함
서비스는 좋다고 하지만 브랜드 임팩트 부족
일부는 좋아하지만 일관성이 없음
대체로 긍정적이지만 더 강한 인상 필요
```

Q7. 이 문제를 해결하는 데 쓸 수 있는 여유는 어느 정도인가요? (단일선택)
```
당장 시작할 준비가 되어있다
3개월 안에 움직일 수 있다
방향만 잡히면 바로 실행 가능하다
아직 탐색 중이다
솔직히 지쳐있다. 그래도 뭔가 바꾸고 싶다
```


### 결과 화면

```
진단이 완료되었습니다.
지금 당신의 브랜드 유형이 분석되었습니다

[유형 타이틀]
[유형 설명]
주요 개선 방향 (4개)

[결 스캔 — BASIC으로 이어서 보기]
[다시 진단하기]
```

#### **결과 유형 3개**

**신생 브랜드형**
설명: 기본기부터 차근차근 쌓아올려야 하는 단계입니다.
주요 개선 방향:
```
브랜드 아이덴티티 구축이 최우선
명확한 타겟 고객 정의 필요
핵심 메시지 개발 집중
일관성 있는 비주얼 시스템 구축
```

**성장 브랜드형**
설명: 기본은 갖췄지만 차별화와 강화가 필요한 단계입니다.
주요 개선 방향:
```
브랜드 포지셔닝 명확화
경쟁사 대비 차별화 전략
브랜드 경험 일관성 강화
디지털 브랜딩 확장
```

**완성 브랜드형**
설명: 기본기는 탄탄하지만 혁신과 진화가 필요한 단계입니다.
주요 개선 방향:
```
브랜드 리뉴얼 또는 리포지셔닝
새로운 시장/고객층 공략
브랜드 확장 전략
AI 활용 브랜딩 자동화
```
