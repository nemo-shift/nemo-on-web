## 🎯 핵심 과제: 완성된 V4.3 뼈대 위의 시각적 정밀 보정

이제 히어로에서 CTA까지 이어지는 기술적 통로는 모두 개통되었습니다. 내일은 설치된 **레드 가이드라인**을 나침반 삼아 인터랙션의 미학적 최고점을 찍는 작업을 수행합니다.

### [ ] 1. 히어로 및 입구 구간 '영점 조절' (Zeroing)
- **1px Sync**: 히어로 [결] 박스와 `SharedNemo`의 합체가 일어나는 시점의 좌표를 1px 단위로 보정.
- **Logo Transition**: 로고가 헤더로 안착할 때의 곡선 곡률과 가속도를 기획서 수준으로 고도화.

### [ ] 2. 레드 가이드라인 기반 '전이 리듬' 튜닝
- 각 섹션 시작선(Red Line)이 화면 상단에 닿는 찰나, 로고의 변형과 배경색 전환이 **기분 좋게 싱크**되도록 타임라인 레이블 수치 재조정.
- **T -> + 모핑**: 모핑 가속도(Ease)를 조정하여 '전환'의 상징성을 극대화.

### [ ] 3. 껍데기만 있는 섹션 '디자인 주입'
- `BrandStory`, `CTA` 등 현재 이름만 있는 섹션들에 기획안의 레이아웃과 이미지를 실제 구현.
- 각 섹션 내부의 텍스트 리빌(Reveal) 타이밍을 스크롤 진행도에 맞춰 설계.
